# 프로젝트 소개

이 프로젝트는 HTML의 `<table>` 태그를 활용하여  
다양한 형태의 데이터 레이아웃을 제작하고 구조를 이해하기 위해 진행한 실습 프로젝트입니다.

단순한 표 제작뿐 아니라,

- 영화 정보
- 레시피 구성
- 시간표
- 동물 정보
- 도서 목록
- 박스오피스 데이터

등 실제 콘텐츠를 기반으로 테이블 UI를 제작했습니다.

# 주요 페이지

## 1. Animal Table

동물 정보를 표 형식으로 정리한 페이지

### Features
- 행 / 열 구성 이해
- 기본 table 구조 학습
- 데이터 정렬 실습

---

## 2. Book Table

도서 목록을 테이블 UI로 구성한 페이지

### Features
- 제목 / 저자 / 가격 구조화
- 가독성 있는 데이터 배치
- 반복적인 데이터 UI 제작

---

## 3. Box Office Table

영화 박스오피스 데이터를 정리한 페이지

### Features
- 순위 기반 데이터 구성
- 숫자 데이터 정렬
- 콘텐츠 중심 테이블 제작

---

## 4. Movie Table

영화 정보를 정리한 테이블 페이지

### Features
- 영화 정보 시각화

- 다양한 셀 구성 연습

## 5. Recipe Table

레시피 순서를 표 형식으로 제작한 페이지

## 6. Timetable

# 프로젝트를 통해 배운 점

이번 프로젝트를 통해 다음 내용을 학습했습니다.

- HTML Table 구조 이해
- `<tr>`, `<td>`, `<th>` 활용
- `rowspan`, `colspan` 사용법
- 데이터 기반 UI 설계
- 반복적인 레이아웃 구성 방식
- 웹 문서 구조화 기초
