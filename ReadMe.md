## 🚀 Project Overview
# 프로젝트 소개

이 프로젝트는 HTML의 `<table>` 태그를 활용하여  
다양한 형태의 데이터 레이아웃을 제작하고 구조를 이해하기 위해 진행한 실습 프로젝트입니다.

단순한 표 제작뿐 아니라,

- 영화 정보
- 레시피 구성
- 시간표
- 동물 정보
- 도서 목록
- 박스오피스 데이터

등 실제 콘텐츠를 기반으로 테이블 UI를 제작했습니다.

<br>


### 🎯 핵심 목표
* **직관적인 사용자 경험:** 사용자가 헤매지 않는 직관적이고 편리한 네비게이션 설계
* **감각적인 비주얼 구성:** 콘텐츠가 돋보일 수 있는 레이아웃과 일관된 비주얼 시스템 구축
* **웹 표준 및 최적화:** 웹 표준을 준수하는 깔끔한 마크업 및 최적화된 배포 프로세스
